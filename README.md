# Student Management System

A beginner-friendly **Student Management System** built in Python that allows users to manage student records through a command-line interface. The application stores student data in a text file, making records available even after the program is closed.

## Features

* Add new student records
* Search students by name
* View all student records
* Delete student records
* Calculate the average marks of all students
* Assign grades based on marks
* Persistent data storage using file handling
* Menu-driven command-line interface

## Technologies Used

* Python 3

## Python Concepts Used

* Variables and Data Types
* Dictionaries
* Loops (`while`, `for`)
* Conditional Statements (`if`, `elif`, `else`)
* Pattern Matching (`match-case`)
* File Handling (`read`, `write`, `append`)
* Exception Handling (`try-except`)


Created as a Python practice project.
